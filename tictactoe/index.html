Tic-Tac-Toe

The classic game of Tic-Tac-Toe in an electronic form! In October 2020, Ben and his friend built this game, and we had great time making this.

You can download the game here. Please direct any questions or comments to tictactoe@java.benchase.info.

Documentation

Variables
Scanner in - Scanner object which allows for the player to input the space to make their move.
grid [] - ArrayList for the positions on the board, values 0-8.
win - Used to set if player won or not. Defaults to 0 (none), 1 = win, 2 = lose
choice - Used to set which player's turn it is. Defaults to 1 (player), 2 = computer
position - Random number generated by the computer to determine where it's move is played

Methods
printgrid() - Displays the board after every turn
